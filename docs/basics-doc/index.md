# مرحبًا

مرحبًا بك في قسم الأساسيات, هنا تجد ما تحتاجه من المعلومات للبدء في مسيرتك البرمجية مرتبة على عدة أقسام:

* التعريف: فيه نبذة هامة عن علم البرمجة وكيف يقرأ الحاسوب النصوص التي نكتبها.

* لغات البرمجة: فيه التعريف بمفهوم لغات البرمجة وأنواعها والفرق بينها.

* مجالات البرمجة: نستعرض فيه أبرز أقسام ومجالات البرمجة الشهيرة مع نبذة عن كل واحد منها.
