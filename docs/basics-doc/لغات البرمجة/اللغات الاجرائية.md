## نبذة تعريفية

تعد اللغات الاجرائية من أهم ركائز البرمجة, وهي ليست لغات برمجة بذاتها وإنما تستعمل مع لغات البرمجة كعامل مساعد. ونستطيع أن نعرّفها بأنها لغات تستعمل لغرض الترتيب, وبيان هيكلية موقع \ مستند ما. تشتمل على ذكر النصوص والعناصر فقط.

## أبرز اللغات

* HTML: تُستعمل مع مواقع الويب لتطوير واجهات المستخدمين.

* XML: وهي شبيهة بلغة ( HTML ) إلا أن استخدامها أوسع فهي تستعمل مع لغات برمجة متنوعة ولا نستخدمها في واجهات المواقع وإنما تستخدم في خزن البيانات الأساسية كإعدادات تطبيق ما.

ويشابهها في الوظيفة لغات ( YAML, TOML ).

* JSON: وهي واسعة الاستعمال تستخدم في مجال الويب مع لغة ( JavaScript و PHP ) وتستعمل أيضًا مع باقي لغات البرمجة في جميع المجالات مثل ( Python, Java, C# ) وغيرها. فكل لغة تقريبا لديها مكتبة أوامر يمكنها من خلالها قراءة ملفات ( JSON ) والتعامل مع البيانات المخزونة فيها.
