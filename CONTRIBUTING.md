# المساهمة في عُلِم
ابتداءً, نشكركم على اهتمامكم في هذا المشروع والمساهمة فيه.

إن **عُلِم** مشروع مفتوح المصدر يمكن للجميع مشاهدة نصّه البرمجي والتعديل عليه وإنشاء نسخ مختلفة منه.
كما ويمكنكم المساهمة في المشروع بعدة طرق:

## المساهمة المعنوية
لو أعجبك المشروع ولا تستطيع المساهمة البرمجية, يمكنك وضع نجمة ( Star ) للمشروع مما يسهم في وصوله وفي إبداء الاهتمام.

## المساهمة بالتجربة
يمكن للجميع فتح الموقع وتجربته من خلال [الرابط](https://binhunaish.github.io/olem.ar/) والإبلاغ عن أي خلل أو إقتراح في خانة ( Issues ) على منصة Github.

## المساهمة البرمجية
يمكنك المساهمة في تطوير المشروع أو المساعدة في كتابة المحتوى بالخطوات الآتية:

### المتطلبات:
*  بيئة تشغيل `Node Js`.
*  برنامج `git`.

1- قم بعمل نسخة من المشروع على حسابك ( Fork ) ثم اسحب المشروع إلى جهازك من خلال كتابة الأمر:
```bash
git clone https://github.com/binhunaish/olem.ar.git
```

2- قم بالدخول إلى المجلد الذي يحوي المشروع:
```bash
cd olem.ar
```

3- قم بتحميل الملفات اللازمة للتشغيل:
```bash
npm i
```

الآن أصبحت جاهزًا لتشغيل الموقع والتعديل على الملفات, يمكنك تشغيل الموقع في أي وقت من خلال التوجه إلى المجلد الذي يحتوي على ملفات المشروع وكتابة الأمر:
```bash
npm start
```

### ما هي الملفات التي يمكنني تعديلها للمساهمة؟

**مهم:** هذا المشروع مبني بواسطة أداة [Docusaurus](https://docusaurus.io/) ويجب على المساهم معرفة طريقة استخدام هذه الأداة ولو بصورة متوسطة.

تستطيع تعديل عدة ملفات للمساهمة في هذا المشروع:
* تستطيع كتابة المحتوى التعليمي والمنشورات في المجلدات المنتهية بـ`-doc` والتي هي المجدات التي تحوي الدروس.
* تستطيع تعديل الشيفرة البرمجية والقالب الخاص بالمشروع باستخدام `react-js`.

**ملاحظة:** يجب أن يكون المحتوى باللغة العربية حصرًا وبأسلوب يجمع الاكاديمية والوضوح.

بعد إنهاء تعديلاتك يمكنك تنفيذ (Pull Request) بالتغييرات التي قمت بها وسيتم مراجعتها من قبل الفريق وقبولها.
